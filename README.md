# DeepStudy — 生成式多模态教学一体管理系统
> 融合文本/图像/语音多模态大模型的智能教学平台

## 简介
面向师生的一站式教学管理系统，基于生成式多模态大模型打造的一站式教学一体化管理平台，整合文本、图像、语音多种交互形式，面向教师与学生双用户群体设计。系统集成智能问答、自动化课件制作、学生学情数据分析、课程信息管理等核心模块，借助AI能力简化教务办公流程、辅助课堂教学、优化学习答疑体验，全方位提升校园教学管理智能化水平。

## 技术架构
- 开发语言：Python，Django
- 服务框架：Web后端开发框架
- 核心技术：生成式大模型、多模态交互、接口开发、数据权限管理、业务功能开发

## 核心功能
1. 智能答疑：依托大模型解答学科知识问题，即时响应学习疑问
2. 课件生成：自动排版生成教学课件，减轻备课工作量
3. 学情分析：统计学习数据，分析学习状态与薄弱知识点
4. 教务管理：统一管理师生信息、课程安排、班级权限
5. 多模态交互：支持文字、语音、图文多样化交互使用
